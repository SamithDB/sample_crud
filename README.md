# SampleCRUD

Technologies : 
    Angular ,
    NodeJS , 
    ExpressJS framework , 
    MS SQL

## Development server
Use folowing commands to run Dev Env,

Run `npm install` 

Run `ng build` 

Run `node server.js`

Please change Data Base config information, before run those commands.
